4.TO PRINT SQUARE SHAPE USING TURTLE

7.TO PRINT ANOTHR PATTERN USING TURTLE
import turtle
colors=["red","purple","blue","green","orange","yellow"]
my_pen=turtle.pen()
turtle.bgcolor("black")
for x in range(360):
my_pen.pencolor(colors[x%6])
